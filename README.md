# Rock Paper Scissors (C++)

A simple command-line implementation of the classic game **Rock Paper Scissors** using C++.

## About the Game
This is a two-player game between the user and the computer.  
Each round, both players choose one of the following options:
- Rock
- Paper
- Scissors

The winner is decided based on the following rules:
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

## Features
- Player vs Computer mode
- Random computer choices
- Score tracking
- Simple and beginner-friendly C++ code

## How It Works
The program:
1. Asks the player to choose Rock, Paper, or Scissors
2. Generates a random choice for the computer
3. Compares both choices
4. Declares the winner of the round
5. Repeats until the player chooses to exit
